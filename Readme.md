# Verziókezelés
## Helyi repo létrehozása
- inicializálás
    > git init 
- ellenőrzés:
    > git status
- előkészítjük commit-re (eltárolni):
    > git add .
- ellenőrzés: 
    > git status
- helyi gépen ellenőrizzük le: username és email
    > git config user.name 

    > git config user.email
- eltároljuk mint egy új verzió:
    > git commit -m "first commit"
